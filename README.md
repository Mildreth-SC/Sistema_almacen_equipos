# Sistema de Gestión de Soporte Técnico — Portotech

API REST desarrollada en Go para administrar el proceso de soporte técnico de equipos tecnológicos en una tienda de cómputo ubicada en Portoviejo.

## Descripción del problema
Actualmente, Portotech registra los mantenimientos, devoluciones y seguimiento técnico de manera manual. Esto provoca pérdida de historial, dificultad para controlar el estado de los equipos y poca comunicación con los clientes.

El sistema busca organizar toda la información relacionada con:

- Órdenes de soporte
- Mantenimientos
- Devoluciones
- Seguimiento técnico

## Tecnologías utilizadas

- Go 1.22+
- Chi Router
- GORM
- SQLite
- PostgreSQL
- Docker
